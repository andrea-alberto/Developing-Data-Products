Goal

The goal of this work is to find correlation between Italy regions GDP and their population data

Data Sources

The ISTAT institution is the source of the data used.

The list of the cities in Italy with their anagraphical data http://demo.istat.it/pop2017/dati/comuni.zip

The administrative structure of Italys cities, to get the regions name of each city https://www.istat.it/storage/codici-unita-amministrative/Elenco-comuni-italiani.csv

The GDP per region http://dati.istat.it/Index.aspx?DataSetCode=DCCN_PILPRODT

Relation between Population and GDP pro capite

The following plot shows the relation between the GDP pro capite and the population of each region

    regioni <- fread("ItalyGDP.csv")
    g1 <- ggplot(regioni, aes(x=Totale.sum,y=get("PIL pro capite (euro)"), color=ZonaBis))
    g1 <- g1 + geom_point() +  geom_smooth(method=lm)
    g1

Regression Model

Here the data of the regression model calculated. As we can see the coefficient of the regression models are not acceptable for the zone Nord and SudIsole.

    model1 <- lm(get("PIL pro capite (euro)") ~ Totale.sum * ZonaBis, data=regioni)
    summary(model1)
